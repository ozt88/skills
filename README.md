# skills

Claude Code 용 개인 스킬 모음.

## 설치

원하는 스킬을 글로벌 또는 프로젝트별로 복사하면 된다.

```bash
# 글로벌
cp -r <skill-name> ~/.claude/skills/

# 프로젝트별
cp -r <skill-name> .claude/skills/
```
