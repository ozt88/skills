# Patterns By Category

Use this file to choose exactly one primary example. Read the selected example directly before writing HTML; palette, typography, layout, and interaction style come from the example.

## Exploration And Planning

- `01-exploration-code-approaches.html`: compare 2-3 approaches, pro/con tables, inline metric badges.
- `02-exploration-visual-designs.html`: compare multiple visual directions with light/dark toggles.
- `16-implementation-plan.html`: milestones, progress badges, risk/mitigation tables, threaded comments.

## Code Review And Code Understanding

- `03-code-review-pr.html`: risk map, file list, inline annotations, severity badges.
- `04-code-understanding.html`: module diagram, call-stack walkthrough, collapsible source.
- `17-pr-writeup.html`: TL;DR, why, file-by-file tour, test plan, rollout notes.

## Design Systems

- `05-design-system.html`: token catalog, type/spacing/shadow hierarchy, live components.
- `06-component-variants.html`: component variant grid with tuning controls.

## Prototypes

- `07-prototype-animation.html`: focused micro-interaction with animation states.
- `08-prototype-interaction.html`: small drag/drop or reorder interaction.

## Diagrams

- `10-svg-illustrations.html`: SVG sheets, visual system diagrams, download actions.
- `13-flowchart-diagram.html`: clickable process flowchart with detail panel.

## Reports And Explainers

- `11-status-report.html`: KPI header, shipped table, timeline or heatmap.
- `12-incident-report.html`: severity metadata, timeline, root cause, impact, actions.
- `14-research-feature-explainer.html`: feature explainer with TOC, staged expansion, FAQ.
- `15-research-concept-explainer.html`: concept walkthrough with simulator, glossary, comparison table.
- `09-slide-deck.html`: slide-like navigation for an existing report or explainer source only.

## Custom Editors

- `18-editor-triage-board.html`: kanban triage, filters, copy-as-markdown export.
- `19-editor-feature-flags.html`: toggles, dependency warnings, copy diff/full JSON.
- `20-editor-prompt-tuner.html`: editable prompt slots, live preview, token count, copy action.
