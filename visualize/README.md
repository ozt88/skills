# visualize

Claude Code skill — render a markdown file or raw content into an interactive HTML page you can preview instantly in VS Code.

## What it solves

Files like `PLAN.md`, `handoff.md`, or anything under `.planning/` are hard to grasp at a glance.

> You: `/visualize PLAN.md`
> Claude: ✅ `.viz/PLAN.html` (table + timeline)

VS Code Live Preview shows it instantly. Re-running the command on the same file auto-refreshes the preview.

## How it works

- **With an argument**: visualizes the given file or raw markdown immediately.
- **Without an argument**: lists recent `.md` files and the last assistant markdown block as candidates, then asks which one to visualize.

The HTML is generated by Claude on the fly — no fixed templates. Hint at a layout if you want: `/visualize PLAN.md as kanban`.

The output UI is translated into the language you're currently using in the conversation (Korean in → Korean UI, English in → English UI, etc.).

## Install

```bash
# Per project
cp -r visualize .claude/skills/

# Or globally
cp -r visualize ~/.claude/skills/
```

Install the **Live Preview** extension (Microsoft) in VS Code. Right-click the generated `.viz/<name>.html` once and choose **Show Preview** — subsequent renders auto-refresh.

## Invocation

- `/visualize PLAN.md` — render that file directly
- `/visualize` — list candidates and ask
- `/visualize PLAN.md as kanban` — render with a specific treatment
